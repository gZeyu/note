- 查询所有数据库名
``` bash
show databases;
```
- 选择数据库
``` bash
use database_name;
``` 
- 创建数据表
```
create table table_name (column_name column_type);
```
- 查询所有表名
``` bash
show tables;
```
- 删除
``` delete语句
delete from table_name [where clause]
```
