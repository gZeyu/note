## 用户注册


### 请求URL：
```
http://localhost:4000/register
```

### 请求方式：
```
POST
```

### 参数类型: 请求体
| 参数 | 是否必选 | 类型 | 说明 |
|--|--|--|--|
| name | Y | string | 用户名 |
| password | Y | string | 密码 |
| phone_number | Y | string | 手机号 |

### 请求示例
```
{
	"name": "tom",
	"password": "123456",
	"phone_number": "18812341234",
}
```
